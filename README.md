 # Train Ticket Booking System (Java JDBC)

A Java-based Train Ticket Booking System developed using Java, JDBC, Servlets, and SQL.
The application supports train search, seat availability, ticket booking, and booking history.

## Tech Stack
- Java (J2EE)
- JDBC
- Servlets
- SQL (Oracle / MySQL)
- Maven
- HTML, CSS

## Features
- User and Admin login
- Search trains and check seat availability
- Online ticket booking
- Booking history
- Admin train management (Add / Update / Delete)

## Purpose
This project is built for learning Java backend development and interview preparation.
